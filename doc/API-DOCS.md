### 1. 获取文件列表

获取指定目录下的所有文件和文件夹列表。

| 项目 | 说明 |
|------|------|
| **URL** | `/main/getAllFile` |
| **方法** | `GET` / `POST` |

**请求参数**:

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `path` | String | 否 | `""` (根目录) | 目标目录路径，相对于存储根目录 |

**响应**: JSON 数组，每个元素结构如下：

```json
{
  "name": "文件名",
  "length": 1024,
  "isFolder": false,
  "lastModified": 1717600000000
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | String | 文件/文件夹名称 |
| `length` | long | 文件大小（字节），文件夹为 0 |
| `isFolder` | boolean | 是否为文件夹 |
| `lastModified` | long | 最后修改时间（Unix 时间戳，毫秒） |

**示例**:

```
GET /main/getAllFile?path=subdir
```

返回：
```json
[
  { "name": "doc.txt", "length": 2048, "isFolder": false, "lastModified": 1717600000000 },
  { "name": "images", "length": 0, "isFolder": true, "lastModified": 1717600000000 }
]
```

### 2. 获取文件信息

获取单个文件/文件夹的详细信息。

| 项目 | 说明 |
|------|------|
| **URL** | `/main/fileInfo` |
| **方法** | `GET` / `POST` |

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `path` | String | 是 | 文件路径，相对于存储根目录 |

**响应**: JSON 对象（结构同上），文件不存在时返回 `{}`。

**示例**:

```
GET /main/fileInfo?path=subdir/doc.txt
```

返回：
```json
{ "name": "doc.txt", "length": 2048, "isFolder": false, "lastModified": 1717600000000 }
```

### 3. 单文件上传

上传单个文件到指定目录。

| 项目 | 说明 |
|------|------|
| **URL** | `/main/fileUpload` |
| **方法** | `GET` / `POST` |
| **Content-Type** | `multipart/form-data` |

**请求参数**:

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `fileName` | MultipartFile | 是 | - | 要上传的文件 |
| `path` | String | 否 | `""` (根目录) | 上传目标目录 |

**响应**: 纯文本

| 响应内容 | 说明 |
|----------|------|
| `上传成功` | 上传成功 |
| `文件为空` | 文件内容为空 |
| `文件名无效` | 无法获取文件名 |

## 通用说明

### 安全机制

- 所有文件操作接口均通过 `safePath()` 方法进行**路径安全校验**，防止路径穿越攻击
- 用户提供的路径参数会自动 URL 解码并过滤，确保最终路径不超出存储根目录
- 文件名参数通过 `File.getName()` 进行安全处理

### 文件下载

三个下载接口（`/download/{name}`、`/get`、`/files/**/{filename}`）底层均调用 `Util.download()` 方法，支持：

- **断点续传**: 支持 HTTP `Range` 请求头，响应状态码 `206`
- **Content-Type 自动识别**: 根据文件扩展名自动判断 MIME 类型
- **编码处理**: 文本类文件自动添加 `charset=UTF-8`

### FileBean 数据模型

所有文件信息接口返回的 JSON 对象结构：

```json
{
  "name": "string - 文件/文件夹名称",
  "length": "long - 文件大小（字节）",
  "isFolder": "boolean - 是否为文件夹",
  "lastModified": "long - 最后修改时间（毫秒级时间戳）"
}
```

### 存储根目录

默认存储路径为 `{工作目录}/file`，可通过启动参数修改：

```bash
java -jar app.jar --path=/custom/storage/path
java -jar app.jar --storage=/custom/storage/path
```
